# cmd4bun

Bun + TypeScript CLI 工具集合，负责辅助 Git 提交信息生成、飞书知识库文档本地同步，以及基于同步索引扫描可疑飞书文档标题。

## 文档阅读顺序

| 顺序 | 文档                       | 什么时候看                                       |
| ---- | -------------------------- | ------------------------------------------------ |
| 1    | docs/{service}/overview.md | 初次进入：服务职责、模块结构、技术栈、上下游依赖 |
| 2    | docs/{service}/business.md | 实现业务逻辑前：状态机、业务规则                 |
| 3    | docs/{service}/flows.md    | 评估影响范围：执行流程、事件级联链路             |

## 规则文件（.claude/rules/，按需加载）

| 文件                                                 | 内容                                       | 加载时机           |
| ---------------------------------------------------- | ------------------------------------------ | ------------------ |
| [architecture.md](.claude/rules/architecture.md)     | 设计原则、编码原则、API 约定、完成检查清单 | 编写代码, 生成任务 |
| [build-and-test.md](.claude/rules/build-and-test.md) | 运行、构建、测试命令和规范                 | 编写执行单测       |

## 构建标准

- 单元测试文件代码覆盖率超过 50%
